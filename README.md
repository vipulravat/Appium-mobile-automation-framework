# Appium Mobile Automation Framework

This project demonstrates a simple mobile automation framework using:

- Java
- Appium
- JUnit
- Maven
- Page Object Model (POM)

Tested application:
Android Calculator App

Automated Scenario:
1. Open Calculator
2. Perform addition (2 + 3)
3. Validate calculation flow

Project Structure:
- BaseTest (Driver setup)
- Page Objects
- Test Classes

Purpose:
Demonstrate mobile automation testing using Appium.
